# Gems Data

## Overview
This repository contains data management and analysis tools
for the GEMS system.

## Installation
```bash
# Clone the repository
git clone https://github.com/blongworth/gems-data.git
cd gems-data
```

## Usage
`web.py` contains functions for getting data from the GEMS web API.
